How to use:
1. Unzip and upload these files to your GitHub repo root (or keep as-is).
2. Edit firebase-config.js with your project's web config (from Firebase console).
3. Ensure Firebase Authentication Email/Password is enabled and you have an admin user UID.
4. Open admin.html and sign in (or create account) to view messages and edit site_content collection.
5. Contact form will work if contact.html includes contact.js and calls initContactForm properly.
